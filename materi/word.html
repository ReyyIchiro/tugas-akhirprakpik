Microsoft Word

Microsoft Word, adalah perangkat lunak yang di gunakan untuk pengolahan teks, dokumen, laporan dsb.


1. Shortcut Pada Microsoft Word

  • Bold/Tebal = Ctrl + B
  • Italic/Miring = Ctrl + I
  • Underline/Garis Bawah = Ctrl + U
  • Center/Rata Tengah = Ctrl + E
  • Left/ Rata Kiri = Ctrl + L
  • Right/Rata Kanan = Ctrl + R
  • Justify/Rata Kiri Kanan = Ctrl + J
  • Open/Buka = Ctrl + O
  • Save/Simpan = Ctrl + S
  • Close/Tutup = Ctrl + W
  • Cut/Potong = Ctrl + X
  • Copy/Salin = Ctrl + C
  • Paste/Tempel = Ctrl + V
  • Select All/Pilih Semua = Ctrl + A

2. Drop Cap

Drop Cap, adalah huruf pertama pada paragraf yang memiliki ukuran yang sangat besar dan lebih dari huruf lainnya.

Fungsi Drop Cap, adalah membuat teks agar lebih menarik sehingga bisa memancing perhatian para pembaca tanpa membuat pembaca tersebut merasa terganggu. Namun, dokumen resmi seperti skripsi, makalah atau karya ilmiah lainnya tidak menggunakan drop cap pada penulisan dokumennya.


2.1 Cara Mengaplikasikannya

  1. Pada Tab Insert, Pilih Drop Cap
  2. Lalu Pilih Style Drop Cap Yang Di Inginkan

3. Pembuatan Makalah

Dalam pembuatan suatu makalah, diperlukan struktur pada penyusunannya. Struktur tersebut diantaranya :

  • Cover
  • Kata Pengatar
  • Daftar Isi
  • Daftar Gambar (Jika Ada)
  • Daftar Tabel (Jika Ada)
  • Bab 1 : Pendahuluan
  • Bab 2 : Pembahasan
  • Bab 3 : Kesimpulan
  • Daftar Pustaka

3.1 Pembuatan Daftar Isi

  1. Siapkan Halaman Baru Terlebih Dahulu Pada Tab Insert, Pilih Blank Page
  2. Gunakan Heading 1 Pada Styles Untuk Judul dan Heading 2 Untuk Sub Judul
  3. Atur Style Pada Heading, Klik Kanan Pada Menu Heading, Lalu Pilih Modify
  • Font : Times New Roman
  • Size 16
  • Centered
  • Add To The Styles Gallery
  1. Buka Tab Reference, Pilih Table Of Contents
  2. Pilih Daftar Isi Yang Di Inginkan
  3. Jika Ada Perubahan Pada Konten Makalah, Maka Daftar Isi Perlu Di Update
  4. Klik Daftar Isi, Pilih Update Table, Update Entire Table

3.2 Pembuatan Daftar Gambar

  1. Masukkan Gambar Yang Di Inginkan
  2. Buka Tab Insert, Pilih Pictures, Pilih This Device
  3. Klik Kanan Pada Gambar, Pilih Insert Caption
  4. Beri Label Gambar
  5. Tulis Caption Yang Di Inginkan
  6. Buka Tab Reference, Pilih Insert Table of Figures

3.3 Pembuatan Daftar Tabel

  1. Untuk Memberi Caption Tabel, Blok Tabel
  2. Klik Kanan Pada Tabel, Pilih Insert Caption
  3. Tulis Caption Gambar Yang Di Inginkan
  4. Lalu Buka Tab Reference, Pilih Insert Table of Figures

4. Mail Merge

Mail Merge, adalah suatu fungsi yang digunakan untuk menghasilkan beberapa surat, label, amplop, tag nama, dan lainnya menggunakan informasi yang disimpan dalam daftar, database, atau spreadsheet.


4.1 Cara Mengaplikasikannya

  1. Buka Tab Mailings
  2. Pilih Start Mail Merge
  3. Pilih Step-By-Step Mail Wizard
  4. Pilih Tipe Dokumen Yang Akan Di Buat, Lalu Next
  5. Pada Select Starting Document, Pilih Use The Current Document, Lalu Next
  6. Pada Select Recipients, Pilih Type a New List, Lalu Create
  7. Setelah Selesai Mengisikan Record Maka Akan Terbuat Data Baru Yaitu *.Mdb
  8. Pada Mail Merge Recipients, Klik OK
  9. Pada Tab Mailings, Pilih Insert Merge Field Yang Di Butuhkan
  10. Untuk Menampilkan Hasil, Pilih Review Result
  11. Jika Telah Selesai Pilih Finish & Merge
