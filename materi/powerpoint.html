Microsoft PowerPoint

Microsoft PowerPoint digunakan untuk membuat file presentasi dalam bentuk slide.

PowerPoint berfungsi :

  1. Untuk menunjukan kesiapan dalam melakukan presentasi
  2. Memudahkan audiens untuk memahami pesan yang anda sampaikan
  3. Memudahkan untuk menyampaikan ide, data dan fakta

A. Cara Membuat Presentasi Yang Baik

  1. Memilih Judul Yang Tepat
  2. Sesuaikan Template
  3. Memilih Font
  4. Sederhana Tapi Komunikatif
  5. Gantikan Bullets Dengan Icon
  6. Ganti Timeline Dengan Smartart
